Diccionario de términos

Vinculación con el plan de lectura, escritura e investigación.

API

Pieza de código que permite a dos aplicaciones comunicarse entre sí para compartir información y funcionalidades.

Clase

Bloque fundamental de construcción en los lenguajes de programación orientada a objetos.

FPS

Número de frames que se muestran cada segundo para conseguir el efecto de animación de los elementos gráficos que forman parte de una escena de un videojuego.

Frame

Cada una de las imágenes que forman parte de una secuencia de animación.

Framework

Estructura o conjunto de herramientas y componentes que proporciona una base para desarrollar aplicaciones.

GIT

Software para el control de versiones en el desarrollo de proyectos de software, basado en el uso de estructuras de almacenamiento denominadas repositorios.

GitHub

Plataforma web de desarrollo colaborativo que permite el alojamiento de repositorios de Git facilitando la colaboración de varios desarrolladores en un mismo proyecto. Cada cuenta de usuario dispone de funcionalidad adicional como la gestión organizaciones, creación de wikis o realización de forks entre otras.

GUI (Gaphical User Interface)

Interfaz gráfica de usuario: conjunto de componentes gráficos (botones, cuadros de texto, menús, listas desplegables, etc) de la interfaz de usuario con los que los usuarios y usuarias interactúan para el manejo de una aplicación.

Java

Lenguaje de programación y plataforma de desarrollo de software creada inicialmente por Sun Microsystems.

Java 2D

Conjunto de recursos para el dibujo de gráficos en dos dimensiones incluido en el API de Java como parte del framework Java Foundation Classes (JFC). 

Java Swing

Kit de recursos para el desarrollo de aplicaciones con interfaz gráfica de usuario incluido en el API de Java como parte del framework Java Foundation Classes (JFC).

Repositorio

Estructura de almacenamiento en la que se guardan las diferentes versiones del código fuente y otros ficheros que formen parte de un proyecto de desarrollo de software.
