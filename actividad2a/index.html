Descripción de la SdA

Etapa educativa

Formación Profesional

Nivel

Primer curso del C.F.G.S. Desarrollo de Aplicaciones Multiplataforma

Agrupamientos y organización espacial

Agrupamiento en grupos de dos o tres integrantes en el salón de clase.

Temporalización

Seis sesiones de dos horas y dos sesiones de una hora, distribuidas en un periodo de dos semanas.

Productos a desarrollar

Infografía y videojuego desarrollado con Java SE.

Herramientas tecnológicas.

Java SE (jdk)

Entorno Integrado de Desarrollo (IDE), uno de los siguientes: Eclipse, NetBeans, IntelliJ Idea o VS Code.

Sistema de control de versiones de Git y repositorio remoto en GitHub.

Herramientas para crear infografías: Designer, VistaCreate, Adobe Express, Canva.

Impacto esperado

Fortalecimiento de competencias técnicas y habilidades interpersonales, como la comunicación y colaboración en equipo.
